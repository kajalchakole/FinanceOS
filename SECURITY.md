# Security Policy

## Supported Versions

Security fixes are applied to the latest `main` branch state.

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Instead:
1. Email the maintainers with a clear description of the issue.
2. Include reproduction steps and potential impact.
3. Share proposed mitigations if available.

Expected response:
- Initial acknowledgment within 72 hours
- Triage update within 7 days

We will coordinate disclosure timing with reporters whenever possible.
