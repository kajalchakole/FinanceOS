# LICENSING

## Overview

FinanceOS follows an **open-core** licensing strategy:

- The open-source core is licensed under **GNU Affero General Public License v3.0 (AGPL-3.0)**.
- Additional commercial licensing options may be offered in the future for organizations that need different terms.

## FinanceOS Open Source License (AGPL-3.0)

The FinanceOS repository is licensed under **AGPL-3.0**.

AGPL-3.0 is a strong copyleft license intended for software that may be run over a network. It ensures that improvements made to the AGPL-licensed code remain available to the community.

### What users are allowed to do

Under AGPL-3.0, you are allowed to:

- **Self host** FinanceOS
- **Modify** the source code
- **Fork** the project
- **Run privately** for personal or internal organizational use

## AGPL SaaS clause (network use)

If someone runs FinanceOS as a hosted service and modifies the code, they must publish their modifications under AGPL-3.0 to users interacting with that modified service.

In practical terms: if you change AGPL-covered FinanceOS code and let users access it over a network, those users must be able to receive the corresponding source code of your modified version.

## Future Commercial Licensing

FinanceOS may offer commercial licenses for organizations that want to:

- Embed FinanceOS into proprietary software, or
- Offer FinanceOS as a hosted/commercial service **without** AGPL copyleft obligations.

Commercial terms (when available) will be provided separately and will coexist with the AGPL open-source license for the community edition.
