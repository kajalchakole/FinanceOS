# FinanceOS Roadmap

This roadmap is directional and may evolve with community feedback.

## Near term (0.x)

### Platform and deployment
- Docker deployment profile (single-command local stack)
- Environment templates and hardening defaults
- Release tagging and upgrade notes

### Data and intelligence
- Portfolio timeline (historical snapshots and trend playback)
- Sector exposure analysis across holdings
- More broker integrations (API and import-driven)
- Better import diagnostics and reconciliation tools

### Planning and simulation
- FIRE simulator (financial independence planning)
- Goal scenario planner with stress cases
- Expanded retirement projection engine

## Mid term (1.x)

### Automation and operations
- Wealth Autopilot recommendations
  - rebalancing guidance
  - cash allocation prompts
  - goal funding prioritization
- Rule-based alerts and digest summaries

### Extensibility
- Plugin system for custom modules and data connectors
- Public extension APIs and event hooks

### Product surface
- Optional cloud database support
- Improved multi-device UX
- Guided onboarding and migration tools

## Long term

- Hosted version with privacy-preserving architecture
- Community plugin marketplace (open-source first)
- Advanced analytics packs and explainable intelligence layers
